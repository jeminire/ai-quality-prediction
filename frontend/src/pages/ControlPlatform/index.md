# 一体化智能管控平台

集成数据管理与AI质量预测的全流程智能管控平台。

## 功能特点

- **数据统一管理** - 工艺参数、物料批次、质量标签、设备状态、环境参数一站式管理
- **AI智能预测** - 基于机器学习的实时质量预测
- **预警通知系统** - 实时预警监控，支持严重预警、风险提醒和系统通知
- **报告自动生成** - 一键生成质量报告、预测报告、趋势分析报告、根因分析报告
- **预测趋势可视化** - SVG折线面积图展示历史预测趋势

## 集成模块

### 1. 数据管理模块
- 工艺参数管理 - 10项工艺参数（加热温度、成型压力、主轴转速等）
- 物料批次管理 - 物料批次追踪与管理
- 质量标签管理 - 质量检测结果与标签
- 设备状态监控 - 设备运行状态实时监控
- 环境参数记录 - 生产环境参数记录

**功能支持：**
- ✅ CSV数据导入
- ✅ CSV数据导出
- ✅ 数据删除
- ✅ 分页浏览（每页20条）
- ✅ 数据筛选过滤

### 2. AI质量预测模块
- 实时质量预测 - 输入工艺参数和/或选择物料批次获取实时预测
- 批量预测 - 批量处理大量预测任务
- 根因分析 - 基于SHAP值分析质量问题的根本原因
- 预测历史记录 - 完整的预测历史追踪
- 预测趋势图表 - 可视化展示预测趋势

**物料批次预测功能：**
- 支持在预测时选择物料批次（可选）
- 物料批次信息包含：供应商、材料类型、化学成分（碳、硅、锰、磷、硫）、硬度、抗拉强度等
- 风险物料会提高缺陷概率预测值
- 预测结果会显示物料批次信息和风险调整值

### 3. 预警通知模块
- 未读消息计数提醒
- 三种通知类型：
  - 🔴 严重预警（缺陷概率≥70%）
  - 🟡 风险提醒（缺陷概率30%-70%）
  - 🔵 系统通知
- 标记已读/全部已读
- 删除通知

## 一体化功能

- 数据录入 → 模型预测 → 根因分析 → 报告生成
- 全流程可视化追踪
- 统一数据视图

## 报告生成功能

支持一键生成以下报告：
- 📊 质量报告（CSV格式）
- 📈 预测报告（CSV格式）
- 📉 趋势分析报告（CSV格式）
- 🔍 根因分析与决策建议报告（CSV格式）
- 📋 综合打印报告（HTML格式，可直接打印）

## 技术栈

- **前端框架**: React 18 + TypeScript + Vite
- **样式框架**: Tailwind CSS 3
- **UI组件**: Lucide React
- **后端框架**: Flask 2.2.5
- **数据库**: SQLite
- **机器学习**: scikit-learn
- **模型解释**: SHAP
