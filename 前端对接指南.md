# 前端对接指南

## 🎯 快速开始

### 1. 服务地址
```
http://localhost:5000/api/v1
```

### 2. 初始化流程（前端页面加载时执行一次）

```javascript
// 初始化数据和服务
async function initializeService() {
  try {
    // 1. 导入工艺参数数据
    await fetch('/api/v1/data/import/process', {
      method: 'POST'
    });
    
    // 2. 导入质量标签数据
    await fetch('/api/v1/data/import/quality', {
      method: 'POST'
    });
    
    // 3. 检查是否已有模型，没有则训练
    const modelRes = await fetch('/api/v1/models/latest');
    const modelData = await modelRes.json();
    
    if (!modelData.data) {
      // 训练新模型
      await fetch('/api/v1/models/', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({ name: 'quality_model', version: '1.0.0' })
      });
    }
    
    console.log('服务初始化完成');
  } catch (error) {
    console.error('初始化失败:', error);
  }
}
```

### 3. 预测功能

#### 3.1 单条预测（仅工艺参数）

```javascript
// 单条预测 - 仅使用工艺参数
async function predict(features) {
  const response = await fetch('/api/v1/predict/', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ features })
  });
  return await response.json();
}

// 特征参数示例:
const features = {
  heating_temperature: 850.5,    // 加热温度 (°C)
  forming_pressure: 120.3,       // 成型压力 (MPa)
  spindle_speed: 1500.0,         // 主轴转速 (rpm)
  coolant_flow: 25.2,            // 冷却液流量 (L/min)
  vibration_amplitude: 2.5,      // 振动幅度 (mm/s)
  current_intensity: 45.1,       // 电流强度 (A)
  mold_temperature: 180.5,       // 模具温度 (°C)
  feed_rate: 300.0,              // 进给速度 (mm/min)
```

#### 3.2 单条预测（工艺参数 + 物料批次）

```javascript
// 单条预测 - 同时使用工艺参数和物料批次
async function predictWithMaterial(features, material) {
  const response = await fetch('/api/v1/predict/', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ features, material })
  });
  return await response.json();
}

// 物料批次参数示例:
const material = {
  material_batch_id: "MBATCH-0008",
  supplier: "C",
  material_type: "QT500",
  carbon_content: 3.454,
  silicon_content: 2.077,
  manganese_content: 0.63,
  phosphorus_content: 0.058,
  sulfur_content: 0.014,
  raw_hardness: 167.8,
  tensile_strength: 472.7,
  material_status: "RISK",
  risk_reason: "硅含量偏低",
  storage_temperature: 23.7,
  storage_humidity: 48.7
};

// 响应示例:
{
  prediction_id: 2,
  prediction: 0.32,              // 缺陷概率
  confidence: 0.85,
  material_batch_id: "MBATCH-0008",
  material_risk_adjustment: 0.15  // 物料风险调整值
}
```

**物料风险调整说明:**
- `material_risk_adjustment` 表示物料批次带来的额外风险调整值
- 当选择风险物料时，预测的缺陷概率会相应提高
- 物料风险调整因素包括：物料状态（RISK状态会增加风险）、供应商（供应商C风险较高）、材料类型（HT300、QT500、QT450等不同材质有不同风险等级）、化学成分（碳、硅、磷、硫含量过高会增加风险）

## 🎯 错误处理示例

```javascript
async function safeRequest(url, options = {}) {
  try {
    const response = await fetch(url, options);
    const result = await response.json();
    
    if (result.code !== 200) {
      console.error('请求失败:', result.message);
      return { success: false, error: result.message };
    }
    
    return { success: true, data: result.data };
  } catch (error) {
    console.error('网络错误:', error);
    return { success: false, error: error.message };
  }
}

// 使用示例
const result = await safeRequest('/api/v1/models/');
if (result.success) {
  console.log('模型列表:', result.data);
} else {
  console.log('获取失败:', result.error);
}
```
